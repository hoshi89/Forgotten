	wait TIME
	* Pause the script processing *

	TIME - Seconds to wait before moving on to the next line in the script

	/////////////////////////////////////////////////////////////////////////////////////

	showtext XPOS YPOS TIME ROWBREAK STYLE color color color SIZE TEXT
	* Display text on the screen *

	XPOS - X coordinate on the screen
	YPOS - Y coordinate on the screen
	TIME - Seconds to display the text, if 0 then text will be displayed until Mouse1 is pressed
	ROWBREAK - Number of characters before breaking line
	STYLE - Italic/Bold/Normal/Underline/Center [IBNUC] (See example.)
	color - Red (0-255)
	color - Green (0-255)
	color - Blue (0-255)
	SIZE - Font size
	TEXT - The text to display

	Example, red, centered, italic text, XPOS doesn't matter if C is in style.
	showtext 150 200 4 50 IC 255 0 0 17 Hello, this is my awesome text!
